# 安装

# 基础

  1. 创建 package.json，并补充以下信息 👇
{
    "scripts": {
        "dev": "nuxi dev",
        "build": "nuxi build",
        "start": "nuxi build && node .output/server/index.mjs"
    },
    "devDependencies": {
        "nuxt3": "latest"
    }
}
  1. 在根目录下创建 app.vue
<!-- app.vue -->
<template>
    <div>Hello, World!!</div>
</template>
  1. 安装依赖
yarn

npm i
  1. 启动项目 (开发环境)
yarn dev

npm run dev
  1. 打包项目
yarn build

npm run build
  1. 启动项目 (生产环境)
yarn start

npm run start


# 类型声明

如果你想要开一个 ts 项目

请在根目录下创建 tsconfig.json，并补充 👇

{
  "extends": "./.nuxt/tsconfig.json"
}

为了更好的类型提示，还需要在根目录下创建 nuxt.config.ts，并补充 👇

import { defineNuxtConfig } from 'nuxt3'

export default defineNuxtConfig({
	typescript: {
		shim: false
	}
})
