# 创建项目

首先需要创建一个 vite 项目

npm create vite

yarn create vite

pnpm create vite


# 进入项目

创建项目之后是在项目之外，我们需要进入到项目中。

TIP

请将以下的 project 替换为你的项目名称。

cd project

或者用 vscode 打开

code project


# 安装依赖

创建完的 vite 项目是未安装 node_modules 依赖的，所以需要安装依赖

npm i

yarn

pnpm i


# 运行项目

最后可以运行下项目看看有没有问题

npm run dev

yarn dev

pnpm dev
